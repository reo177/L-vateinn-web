# Lævateinn Web Site

Web公開用のHTMLファイルです。

## ファイル構成

- `index.html` - メインHTMLファイル
- `favicon.png` - サイトアイコン（このファイルを配置してください）
- `site.webmanifest` - PWA用マニフェストファイル

## 使用方法

1. `favicon.png` ファイルを `web` フォルダに配置してください
2. `index.html` 内のOGP画像URLやサイトURLを実際のURLに変更してください
3. Webサーバーにアップロードしてください

## 注意事項

- favicon.pngは512x512px推奨です
- OGP画像も同じfavicon.pngを使用するか、別途1200x630pxの画像を用意してください
